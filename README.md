# Weather Bot

## Setup

Copy token from BotFather and OpenWeatherMap to `.env` file
```dosini
BOT_TOKEN=your_bot_token
OWM_TOKEN=your_openweathermap_token
```

## Run

```bash
go run -mod=vendor cmd/main.go
```
